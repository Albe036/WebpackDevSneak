<h1>Bajar proyecto y ejcutarlo como desarrollo</h1>
<ol>
  <li>Copiar enlace de repositorio e ir a la consola del sistema, "GIT CLONE URL", (debes tener instalado git)</li>
  <li>Luego de obtener los archivos clonados, ejecutar el comando "GIT INSTALL"</li>
  <li>PUEDES VER LOS COMANDOS EN EL ARCHIVO "PACKAGE.JSON", pero puedes ejecutar "NPM RUN START:DEV" para abrirlo con Webpack dev server</li>
  <li>Se abrira una una pestaña en el navegador y podras visualizar la web y realizar cambios al codigo.</li>
  <li>Gracias.</li>
</ol>
